Projects

Number Guessing Game

The Number Guessing Game is a fun and interactive web application where users can test their guessing skills. The game randomly selects a number within a specified range, and the player must guess the number. With each guess, the game provides feedback indicating whether the guess was too high, too low, or correct. The goal is to guess the number with the fewest attempts possible.

Key Features:

  • Random number generation within a defined range.
  • User-friendly interface with real-time feedback.
  • Keeps track of the number of guesses taken.
  • Option to reset the game and start over.

Technologies Used: HTML, CSS, JavaScript

Random Quote Generator

The Random Quote Generator is a web application that provides users with a new inspirational quote at the click of a button. Each quote is displayed with the author's name, offering users a source of motivation and reflection. The application also includes a mobile version, allowing users to access their favorite quotes on the go.

Key Features:

  • Displays a new random quote with each click.
  • Quotes are accompanied by the author's name.
  • Clean and responsive design for both web and mobile versions.
  • Easy to share quotes on social media platforms.

Technologies Used (Web): HTML, CSS, JavaScript

Technologies Used (Mobile): Flutter Framework, Dart

Find a Car Ride (Andriod App)

The Find a Car Ride app is an andriod application designed to help users find car rides quickly and efficiently. Whether you need a ride for your daily commute or a spontaneous trip, this app connects you with available drivers in your area. The app features a user-friendly interface, real-time ride tracking, and secure payment options, making it a convenient solution for transportation needs.

Key Features:

  • User registration and profile management.
  • Real-time location tracking and ride updates.
  • Secure in-app payment system.
  • Ratings and reviews for drivers and passengers.
  • Easy ride booking and cancellation.

Technologies Used: Swift, Kotlin, Firebase for backend services
